# Exam Tracker

Exam Tracker is a Node.js/Express app for tracking exam digitization progress, managing related files, and generating reports.

## Start Here

- Main documentation entry: `docs/INDEX.md`
- Quick project summary: `PROJECT_OVERVIEW.md`
- AI chat/project workflow rules: `docs/AI_CHAT_INSTRUCTIONS.md`
